# Demo

### [C++] GUI 示例

`gui_demo.cpp`。运行需要 [mLib](github.com/masterLazy/mLib/)。

### [C++] 数据集制造示例

`dataset_maker_demo.cpp`

### [C++] pybind11 绑定示例

`bind_demo.cpp`。仓库中附的 `gobang.pyd` 是通过此编译得到的。

### [Python] 数据集加载示例

`gobangdataset.py`。其提供了一个 PyTorch 数据集加载接口 `GobangDataset`。你可以通过这个接口加载并扩展本仓库提供的数据集。
